NEXUS PARAGUAY - SITE COM GITHUB PAGES

Arquivos:
- index.html
- imoveis.json

Passo a passo:
1. Crie uma conta no GitHub ou entre na sua conta.
2. Crie um repositório público chamado nexus-paraguay.
3. Envie os arquivos index.html e imoveis.json para o repositório.
4. Vá em Settings > Pages.
5. Em Source, escolha Deploy from a branch.
6. Branch: main / root. Clique Save.
7. O GitHub vai gerar um link grátis do site.
8. Crie um Fine-grained token em GitHub > Settings > Developer settings > Personal access tokens.
9. Permissão: Contents = Read and write apenas nesse repositório.
10. Abra o site, clique Área restrita, senha: nexus786182.
11. Coloque usuário GitHub, repositório e token.
12. Cadastre/edite/oculte imóveis e clique Salvar online.

Observação: imagens grandes deixam o imoveis.json pesado. Use fotos comprimidas.
